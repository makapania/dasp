This directory contains archived files that are no longer needed in the main project directory but are preserved for historical reference.

## Structure

### `tests_archive/`
Validation and test files from development sessions:
- Sprint validation tests (test_sprint1-4_validation.py)
- Boosting model tests
- Parameter validation tests
- Manual validation scripts
- Test documentation

### `documentation_archive/`
Summaries and handoff documents from completed work:
- Model fix summaries (Boosting, GUI fixes, etc.)
- Implementation summaries (Subtabs, Neural/SVM, UX improvements)
- Hyperparameter planning and analysis documents
- UI styling handoff documents

### `2025-11-12/`
Planning documents and code snippets from November 12, 2025 session:
- Agent execution instructions
- Random Forest parameter extraction
- Phase completion instructions

## Note

These files are preserved for reference but are not essential to current development. Active work should focus on files in the main project directory.
